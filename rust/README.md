# bowling_game_kata : rust

Rust で ボーリングのスコア計算

## ビルド方法

``` powershell
cd .\bowling_game
cargo build
```

## ビルドして実行

``` powershell
cd .\bowling_game
cargo run
```

``` powershell
cd .\bowling_game
cargo run ..\..\testdata\all_ones.txt
```

## テスト実行

``` powershell
cd .\bowling_game
cargo test
```
